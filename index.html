About Our Project & Report

Project Description

Naqaa Store is an advanced e-commerce web platform specialized in displaying and managing cosmetics and skincare items. Designed responsively for all devices using HTML5, CSS3 (Flexbox & Media Queries), and pure JavaScript, the site processes full data operations dynamically through Local Storage, achieving seamless interactivity without relying on backend databases.
